> 还没有来得及让他庆幸自己有了阴阳眼，几个突然出现的神灵就开始给阿辽规划起了人生。林家的守护神林虎希望阿辽能在毕业后回老家从事神职，原住民部落的云豹神灵李克劳则相信阿辽是部落新祭司的最佳人选。就连学校舞蹈社的学长颜书齐，也被不知道从哪里来的石虎精给附了身，一直拖着阿辽想让他陪自己为一个乡下小庙复兴香火。还没搞清楚到底发生了什么的阿辽，就这样被推进了不一样的世界线。
